Проект по ЯЛ
Тема: «Конвертер валют»

Описание:
	Целью данного проекта является разработка приложения-конвертера валют, которое позволит пользователю быстро и удобно переводить одну валюту в другую на основе актуальных курсов обмена. Приложение предназначено для людей, которым необходимо конвертировать различные валюты для путешествий, покупок, ведения бизнеса и других финансовых операций.

ТЗ:
1. Создать интерфейс для приложения используя QtDesigner
2. Написать основную функцию конвертации выбранных валют, добавить обновление курса по данным из интернета
3. Добавить выпадающий список с возможными режимами использования программы (обычный, для аналитики, история операций, настройки)
4. Добавить возможность выбора промежутка времени для вывода и анализа графика за это время (режим аналитики)
5. Сохранять операции в базу данных (история операций)
6. Добавить смену темы на темную/светлую (настройки)
7. Создать EXE файл для быстрого запуска приложения
